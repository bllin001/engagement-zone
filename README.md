# Engagement Zone Project

A Streamlit application for analyzing engagement zones and path planning.

## Setup Instructions

### Python Environment

This project requires **Python 3.10.18** and uses a Conda environment named `ez`.


1. **Create the `ez` environment**:
   ```bash
   conda create -n ez python=3.10.18
   conda activate ez
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Streamlit app**:
   ```bash
   streamlit run code/Introduction.py
   ```

### Running the Application

Once your environment is set up, activate it and run:

```bash
conda activate ez
streamlit run code/ez.py
```
